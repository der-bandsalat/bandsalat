# Code of Conduct

Bandsalat ist ein kleines, persönliches Open-Source-Projekt. Wir halten die
Regeln entsprechend schlank.

## Erwartetes Verhalten

- Sei freundlich und respektvoll.
- Geh davon aus, dass das Gegenüber es gut meint.
- Kritik bezieht sich auf Code und Ideen, nicht auf Personen.
- Sprache: Deutsch oder Englisch ist beides ok.

## Nicht erwünscht

- Beleidigungen, Belästigung, persönliche Angriffe.
- Diskriminierende Inhalte oder Symbole.
- Spam, Off-Topic-Werbung.
- Politische Grabenkämpfe, die nichts mit dem Projekt zu tun haben.

## Konflikt-Klärung

Bei Problemen melde dich beim Maintainer (siehe Repo-Profil) per
E-Mail oder über das GitHub Private Vulnerability Reporting.
Der Maintainer entscheidet im Zweifel über Konsequenzen — von einer
Erinnerung bis zum Repo-Block, je nach Schwere.

## Geltungsbereich

Dieser Code gilt in allen Projekt-Räumen: Issues, Pull Requests,
Discussions, Commits, Code-Kommentaren.
